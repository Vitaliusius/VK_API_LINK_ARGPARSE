# Обрезка ссылок и подсчет посещений с помощью Вконтакте.

Эта программа позволяет преобразовать любую сслку в короткую с доменом vk.cc, и подсчитать количество переходов по ней  с помощью API запросов. 

## Окружение

Python3.14.2 должен быть уже установлен.

## Зависимости

Используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

Рекомендуется использовать [virtualemv/venv](https://docs.python.org/3/library/venv.html) для изоляции проекта.

## Переменные окружения

Для отправки API запроса сайту необходим сервисный ключ (токен) приложения. В данной программе используется переменная окружения 'VK_SERVICE_KEY'  которой присваивается сервисный ключ

### Как получить 
Создать мини-приложение Вконтакте по ссылке: [Создание мини-приложений](https://vk.cc/cTNUK9), скопировать сервисный ключ и вставить в файл ".env", присвоив переменной окружения 'VK_SERVICE_KEY', аналогично примеру ниже:
```
VK_SERVICE_KEY = 'a0ef04aca0ef04aca0ef04aca3a3d19'
``` 

## Запуск

Для преобразования ссылки в короткую необходимо запустить файл main.py  с агрументом:
```
main.py https://dev.vk.com/ru/mini-apps/management/creating-new-apps?ref=old_admin_panel
Короткая ссылка:  https://vk.cc/cTNUK9
```
Для подсчета количества перехода по короткой ссылке необходимо запустить файл main.py с аргументом в виде короткой ссылки с доменом vk.cc:
```
main.py https://vk.cc/cTNUK9
Количество кликов:  17
```
